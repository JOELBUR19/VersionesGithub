## Requisitos

Antes de comenzar, asegúrate de tener instalado:

* **Node.js** 
* **npm** (gestor de paquetes de Node)

Puedes verificar si ya los tienes instalados:

```bash
node -v
npm -v
```

Si no los tienes, descárgalos desde: [https://nodejs.org/]

---

## Clonar el repositorio

```bash
git clone https://github.com/JOELBUR19/VersionesGithub.git
```

Luego entra en la carpeta del proyecto:

```bash
cd VersionesGithub
```

---

## Instalar dependencias

Este proyecto utiliza **prompt-sync** para solicitar datos desde la consola.

Para instalar las dependencias ejecuta:

```bash
npm install
```

Esto generará la carpeta `node_modules/`.

---

## Ejecutar un archivo

Desde la terminal, usa:

```bash
node ruta/del/archivo.js
```

### Ejemplos:

```bash
node features/calculator.js
```

```bash
node src/Clase_2/1.ejercicioscsl.js
```

```bash
node src/Clase_2/3.promedioNotas.js
```
